## Ansible Install and configure

- sudo apt install git python3-pip
- pip3 install ansible

- sudo cp /etc/profile ~/.profile
- sudo nano ~/.profile
- add to the end of file: export PATH="/home/{user}/.local/bin:$PATH"
- echo $PATH

- ansible --version
